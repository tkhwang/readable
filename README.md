# Readable

## Prerequite

```bash
npm install -g @nrwl/cli    // nx
npm install -g @nestjs/cli  // nest
```

## Install

```bash
yarn
```

## Run

#### Server

```bash
yarn dev server
```

#### Client

```bash
yarn dev client
```
